Ketra - Android Information Gatherer

Python Android Security
Latest Version: v1.0
Last Updated: 2023
Language: Python

About

Ketra is a powerful Python tool designed for Android penetration testing and information gathering. It provides comprehensive device information collection, system analysis, network scanning, and security assessment capabilities.

Features

  • Device Information Collection
  • System Analysis
  • Network Scanning
  • Security Assessment

Installation

git clone https://github.com/Kcyb3r/Ketra.git
cd Ketra
pip install -r requirements.txt

Usage

python3 ketra.py
