AmazeBOT - Telegram File Manager

Python Telegram Bot
Latest Version: v1.0
Last Updated: 2023
Language: Python

About

AmazeBOT is a versatile Telegram bot built in Python for remote file management on desktop servers. It provides a convenient interface through Telegram for managing files, executing commands, and monitoring your server.

Features

  • Remote File Listing
  • File Download Capabilities
  • Basic Command Execution
  • File Content Viewing
  • Telegram Interface Integration

Installation

git clone https://github.com/Kcyb3r/AmazeBOT.git
cd AmazeBOT
pip install -r requirements.txt

Usage

python3 amazebot.py
