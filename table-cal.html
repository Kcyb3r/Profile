About

Table-cal is a Python-based utility designed for efficient table calculations and data processing. It provides a simple yet powerful interface for handling tabular data and performing complex calculations.

Features

  • Table Data Processing
  • Advanced Calculations
  • Data Export Capabilities
  • User-Friendly Interface

Installation

git clone https://github.com/Kcyb3r/Table-cal.git
cd Table-cal
pip install -r requirements.txt

Usage

python3 table-cal.py
